When Kas

A predictor for when kaspa overtakes bitcoin. It finds when kaspa powerlaw for the kas / btc price surpasses the price for it to be worth more then bitcoin. 

<img width="955" alt="image" src="https://github.com/whenkas/whenkas.github.io/assets/169737990/ff0b622d-0d5b-4c8d-96f0-6d763878cc37">


Data from export here https://coincodex.com/crypto/kaspa/historical-data/, priced in btc. Replace file public/data/kaspa_prices_btc_historical.csv


To run, get data from above, then 'yarn' && 'yarn start'
