When Kas

A predictor for when kaspa overtakes bitcoin. It finds when kaspa powerlaw for the kas / btc price surpasses the price for it to be worth more then bitcoin. 

<img width="953" alt="image" src="https://github.com/whenkas/whenkas.github.io/assets/169737990/fc68a24a-4afb-4d8c-903f-5b6043f7cf20">


Data from export here https://coincodex.com/crypto/kaspa/historical-data/, priced in btc. Select the full time range of kaspa existance, frequency Daily, currency BTC. Export. Replace file public/data/kaspa_prices_btc_historical.csv


To run, get data from above, then 'yarn' && 'yarn start'
